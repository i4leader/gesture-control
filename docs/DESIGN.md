# 设计文档: 粒子文字控制器

## 概述
这是一个利用 MediaPipe 手势识别控制 3D 粒子特效的网页应用。通过特定的手势，用户可以让粒子汇聚成文字或产生爆炸效果。

## 功能描述

### 1. 手势交互
- **展示手掌 (Open Palm)**: 粒子汇聚成 "你好阿里云"。
- **OK 手势**: 粒子变换为 "OK"。
- **挥手 (Wave)**: 粒子变换为 "ByeBye"。
- **爆发 (Sudden Spread)**: 从握拳状态突然张开，粒子向外爆炸飞散。

### 2. 视觉效果
- **粒子系统**: 使用 Three.js `Points` 渲染。
- **发光效果**: 简单的发光材质或后期处理。
- **文字生成**: 动态将文字渲染到 Canvas，采样像素点坐标作为粒子目标位置。

## 技术实现
- **Three.js**: 负责 3D 场景与粒子运动。
- **MediaPipe Hands**: 负责实时手势捕捉。
- **Canvas API**: 用于生成文字的点阵数据。
